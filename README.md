## ¿Qué es **YuhanPicos OS**?

**YuhanPicos OS** es un portafolio web presentado como un *sistema operativo retro* totalmente interactivo.  
Combina nostalgia de los 90 con tecnologías modernas para ofrecer una experiencia única de navegación:

- Ventanas draggable
- Aplicaciones internas
- Animaciones fluidas
- Terminal funcional
- CV exportable y limpio

## Características Principales

### Sistema Operativo Retro  
Interfaz estilo escritorio de los 90 con ventanas que puedes abrir, cerrar, arrastrar y enfocar.

<div align="center">
  <img src="https://i.ibb.co/49wbQzn/image.png" width="600" alt="View Demo"/>
</div>

### Aplicaciones Internas  
- **Sobre Mí**  
- **Proyectos**  
- **Contacto**  
- **Currículum (modo impresión A4)**  

### Terminal Funcional  
Incluye comandos como:

```

help
open projects
about
clear

````

### Modo CV Imprimible  
Diseño minimalista profesional listo para guardar en PDF.

### Responsividad Completa  
Perfectamente usable en móviles, tablets y desktops.

## Instalación

### Prerrequisitos  
- Node.js 18+  
- Git  

### Clonar el repositorio  
```bash
git clone https://github.com/xXYUHANXx/Portfolio.git
cd Portfolio
````

### Instalar dependencias

```bash
npm install
```

### Modo desarrollo

```bash
npm run dev
```

Accede en:
**[http://localhost:9002](http://localhost:9002)**

## Despliegue

Compatible con:

* **Vercel (recomendado)**
* Netlify (con adaptadores)
* Servidores Node.js

No compatible con GitHub Pages
Debido a las funciones del servidor + terminal.
